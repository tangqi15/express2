# express2

express2个人服务器

使用express-generator 脚手架

##

# 1.安装express-generator

```
npm install express-generator -g
```

# 2.创建项目

```
express express2
```

# 3.安装依赖

```
cd express2
npm install
```

# 4.启动项目

```
npm start
```

# 5.访问项目

```
http://localhost:3001

bin/www   改端口号
```

# 6.项目目录结构

```
express2
│
├── app.js
├── bin
│   └── www
├── package.json
